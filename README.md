<h2>Описание проекта</h2>
<p>Реализация модели задачи `Task` с использованием дескрипторов для валидации атрибутов и `@property` для вычисляемых свойств. Работа демонстрирует принципы защиты инвариантов и управление доступом к атрибутам.</p>

<h2>Дескрипторы</h2>
<p>Были реализованы следующие дескрипторы:</p>
<ul>
<li>`IdDescriptor` - валидирует id, запрещает изменение</li>
<li>`DescriptionDescriptor` - валидирует описание, разрешает изменение</li>
<li>`PriorityDescriptor` - валидирует приоритет, разрешает изменение</li>
<li>`StatusDescriptor` - валидирует статус, разрешает изменение</li>
<li>`CreationTimeDescriptor` - non-data дескриптор, автоматически устанавливает время создания, разрешает только обращение</li>
</ul>

<h2>Property</h2>
<p>Property для вычисляемых свойств:</p>
<ul>
<li>`is_ready` - проверяет статус на соответствик ready</li>
<li>`time_in_queue` - вычисляет время с момента создания task</li>
</ul>
